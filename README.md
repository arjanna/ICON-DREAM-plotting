# ICON-DREAM-plotting
Python basic plotting routines for ICON-DREAM data


The scripts
Plotting_T2M_and_grid_EU_nest.py and Plotting_T2M_and_grid_global.py
have been used to create the plots of the README_intro_ICON.pdf, which can be found
in the Climate Data Center of DWD, e.g.

https://opendata.dwd.de/climate_environment/REA/ICON-DREAM-EU/
